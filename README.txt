# 🎬 MultiSite Video & Audio Downloader (GUI)

A Python program with a graphical interface to download videos or audio from multiple websites using **yt-dlp**.

## 🔧 Requirements
- Python 3.8 or higher
- yt-dlp library

## ⚙️ Installation
Run this command to install required packages:
```bash
pip install -r requirements.txt
```

## ▶️ Usage
Start the program using:
```bash
python main.py
```

Then paste any video link from supported sites like:
- YouTube
- TikTok
- Twitter
- Facebook
- Instagram
- SoundCloud
- and many more...

Choose format and quality, then press **Download**.
